MooSweeper
==========

Costumizable Minesweeper clone.

![Screenshot](http://timbaumann.info/MooSweeper/Demo/screenshot.png)

How to use
----------

	#JS
	var mooSweeper = new MooSweeper($$('body')[0], {
		cols: 10,
		rows: 10,
		where: 'bottom' // where the minesweeper is inserted
	});
