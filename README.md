MooSweeper
==========

Costumizable Minesweeper clone.

![Screenshot](http://projects.timbaumann.info/MooSweeper/Demo/screenshot1.png)

How to use
----------

	#JS
	var mooSweeper = new MooSweeper(document.body, {
		cols: 10,
		rows: 10,
		where: 'bottom' // where the minesweeper is inserted
	});
